Problem: Calculate the customer’s monthly cell phone billInput: plan, number of minutesOutput: final billdisplay Plan optionsget planif plan is ‘A’ or ‘a’	if number of minutes > 450		extra minutes = number of minutes – 450		extra bill = extra minutes * 0.45		final bill = 39.99 + extra bill	else		final bill = 39.99else if plan is ‘B’ or ‘b’	if number of minutes > 900		extra minutes = number of minutes – 900		extra bill = extra minutes * 0.40		final bill = 59.99 + extra bill	else		final bill = 59.99else if plan is ‘C’ or ‘c’	final bill = 69.99else	display invalid inputdisplay final billhalt
